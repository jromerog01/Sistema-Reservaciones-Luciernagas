### Apps del Proyecto


- `usuarios`: Esta aplicacion esta pensada para manejar y encargarse de la gestion de los
usuarios del sistema
- `parques`: Su objetivo es el manejo y creacion de todo lo involucrado con los parques, como por ejemplo
  - Hospedajes
  - Servicios de los Parques 
  - Mapa Interactivo
- `reservaciones`: Es el modulo principal del proyecto, ya que aqui se manejara toda la logica relacionada con
la creacion de las reservaciones y todas las verificaciones que esto implica por detras


### Ambiente Miniconda
Se agrega el archivo `environment.yml` para la creacion del ambiente necesario para que este proyecto funcione
correctamente. El comando para crear el ambiente es:
```shell
conda env create -f environment.yml
```
